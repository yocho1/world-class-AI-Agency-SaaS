# PART 2 — UX FAILURE ANALYSIS (AI AGENCY WEBSITES)

For each failure mode below, provide:

- **Impact on conversion** (quantified where possible)

## Failure Modes to Analyze

1. **Fake / useless chatbots** — chatbots that say "Hi! How can I help?" and fail to answer real questions
2. **Vague value proposition** — "We use cutting-edge AI to transform your business"
3. **Weak trust signals** — generic testimonials, no metrics, no named clients
4. **Overdesigned / heavy landing pages** — WebGL backgrounds, 8MB page weight, 6s load time
5. **Poor mobile UX** — desktop-first design shrunk to mobile, tiny tap targets, horizontal scroll
6. **No real product demonstration** — all description, zero demonstration
7. **Single high-friction CTA** — only "Contact Us" with no nurture path
8. **Jargon-heavy navigation** — "AI Enablement Solutions Portfolio"
9. **Stock photo syndrome** — robot hands, glowing brains, people staring at holographic screens
10. **No clear user journey** — treating a startup founder and enterprise CTO identically
